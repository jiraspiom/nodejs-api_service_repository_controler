### FAQ - pergunta frequentes

#### 💬 Como iniciar o banco de dados
execute o comando
```bash
npx prisma migrate dev
```

#### 💬 Como inciar o Bando de dados SQLite
```bash
npx prisma init --datasource-provider SQLite
```

#### 💬 Como ja baixar as tabelad do banco de dados
```bash
npx prisma db pull
```
